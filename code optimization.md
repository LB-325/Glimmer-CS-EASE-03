# 一、传参技巧

***数组传参***：在函数调用时，若以数组为参数传递时，不能在数字名后加[ ]，应再传递一个参数代表数组长度，可以利用此特性，当一个数组中数据没填满或只想传入前面部分数据时，可以控制传入长度，可以减少运算时间，优化了代码
